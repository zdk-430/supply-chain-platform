# 📦 区块链供应链平台 (Supply Chain Platform)

## 📝 项目简介
这是一个基于 Hyperledger Fabric 框架开发的区块链供应链管理平台。旨在利用区块链不可篡改、可追溯的特性，解决传统供应链中数据不透明、信任成本高的问题。

## 🛠️ 核心技术
- **底层框架：** Hyperledger Fabric
- **智能合约（链码）：** Go / Node.js
- **应用开发：** Node.js / Express
- **前端展示：** Vue / React (根据实际情况修改)

## ✨ 核心功能
1. **产品溯源：** 记录商品从原材料到消费者的全生命周期。
2. **节点管理：** 支持供应商、制造商、物流商等多方角色接入。
3. **数据上链：** 确保所有交易和流转记录公开透明且不可篡改。

## 🚀 如何运行
1. 启动 Fabric 测试网络
2. 部署链码
3. 运行后端 API 服务
